# SUDOKULO

## About

* A 9x9 sudoku solver written in Rust that aims to be fast
* Project status: working

## Usage

``` bash
sudokulo --help
```

### Requirements

* [the Rust programming language toolchain](https://www.rust-lang.org/tools/install)

### Build

``` bash
cargo build --release
```

## Run

* for release
    ``` bash
    ./target/release/sudokulo --help
    ```
* for debug
    ``` bash
    cargo run
    ```
